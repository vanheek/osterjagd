APK-BUILD ANLEITUNG (mit Capacitor)
-------------------------------------
1. Voraussetzungen:
   - Node.js + npm
   - Android Studio

2. Installation:
   npm install -g @capacitor/cli
   mkdir osterjagd-app && cd osterjagd-app
   npm init @capacitor/app

3. Projekt einbinden:
   → Kopiere den Inhalt dieses ZIP-Ordners in das Webverzeichnis deines Projekts (z. B. ./www)

4. Plattform hinzufügen:
   npx cap add android

5. APK erstellen:
   npx cap open android
   → Dann im Android Studio: "Build > Build Bundle(s) / APK(s) > Build APK"

Fertig! 🎉
